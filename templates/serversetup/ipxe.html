Hier haben Sie die Möglichkeit, eigenen Menü-Code zum angezeigten PXE-Menü hinzuzufügen, um z.B. auf weitere PXE-Server zu verweisen. Das Format entspricht dem syslinux Menüformat.
Bespiel: 
LABEL my-entry
	MENU LABEL ^Mein Eintrag
	KERNEL http://1.2.3.4/kernel
	INITRD http://1.2.3.4/initramfs-stage31
	APPEND custom=option
	IPAPPEND 3
Sie können ein oder mehrere Einträge erzeugen. Wenn Sie einen Eintrag erzeugen möchten, der automatisch gestartet wird, wenn der Benutzer keine Auswahl tätigt, vergeben Sie als LABEL my-entry und wählen Sie als Standard-Bootverhalten ebenfalls my-entry.
